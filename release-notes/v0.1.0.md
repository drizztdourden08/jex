# Jex v0.1.0

First public release — a local desktop app that mirrors your Steam library and lets
you filter, randomize, and AI-query it, entirely on your machine.

## Features

- **Setup-free start** — auto-detects your Steam install (registry + library folders)
  and shows your installed games with cover art, no key and no network required.
- **Full library mirror** — add your Steam Web API key in Settings to pull your
  complete owned library (including not-installed games), playtime, and rich metadata
  + media into a local SQLite store.
- **AI command bar** — a baked-in local model (no key, no account) turns plain English
  like "a random roguelite with base building I haven't played" into a query over your
  library.
- **Faceted filtering + randomizer** — browse and filter on any metadata, or roll a
  random pick across your library.
- **Glass detail view** — screenshots, trailers, and launch, in the "Aurora" UI.
- **First-run disclosure** — a clear explanation of what runs locally before anything
  touches the network.

## Notes

- Windows-only for now. Installer and portable builds are attached below.
- Everything lives under your OS app-data directory; secrets are encrypted with
  Electron `safeStorage` and sent only to Steam.
- The AI model downloads on first use (one-time, ~2 GB).
